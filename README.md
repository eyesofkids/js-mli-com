# js-mli-com

## 初始化專案

```sh
npm init -y
```

## ESLint

安裝 ESLint:

```sh
npm install eslint --save-dev
```

初始化 ESLint 設定檔:

```sh
npx eslint --init
```

修正 ESLint 錯誤:

```sh
npx eslint --fix .
```
